📋 Spreadsheet Format Guide
Row 1 (header):
Team · Venue Address · Priority · Division · [date] · [date] · …

Priority values: 1 or 2
Division values: A or B

Availability cell values:
Yes — available (home or away)
No — not available
Yes - Away — away games only
Yes - Home — home games only

Date columns: any Excel date format (Apr 2 2026, 4/2/2026, etc.)

Drop XLSX here or click to browse
League schedule Excel file with one tab per team
📋 Format Guide
Sheet names: Must match a team name in the app (e.g. "AYRA", "BlackSox").
Tabs that don't match any team are skipped automatically.

Columns (row 3):
Date — Excel date or text
Day — Thu, Sun, etc.
Time — 17:45, 14:00, etc.
H/A — "Home" or "Away"
Opponent — team name
Game — SINGLE, G1, or G2

Duplicate prevention:
Only Home rows are used to build games, so each matchup is created once regardless of how many team tabs are present.
